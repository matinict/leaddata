# 📺 360Debate — Clean 1-Year Strategy & Monetization Plan

# CHAPTER 1: CORE STRATEGY

## 1.1 Channel Position

* Channel: 360Debate
* Focus (Phase 1): Debate-only
* Expansion (later): Podcast + Classroom via playlists
* Tagline: See Every Side, Decide Smarter

## 1.2 Structure

Single Channel Model:

* Debate (Core growth)
* Podcast (Authority – later stage)
* 360Learn (SEO – later stage)

Rule:
Start with Debate only → Expand after traction

# CHAPTER 2: CONTENT SYSTEM

## 2.1 Debate Format (Primary Engine)

* Structure:

  1. Hook (0–5 sec)
  2. Pro argument
  3. Oppose argument
  4. Judge conclusion
  5. CTA

* Output per topic:

  * 1 HD video (5–10 min)
  * 3–5 Shorts

## 2.2 Topic Categories

* AI
* Study abroad
* Tech vs Jobs
* Finance / Career
* Global comparisons

## 2.3 Upload Frequency

* 4–7 videos/week
* Daily Shorts
